# Emotion Detection Model
This project is a real-time facial emotion detection system built using Python, OpenCV, and TensorFlow/Keras. It is trained on the FER2013 dataset and can recognize human emotions from webcam input by detecting faces and classifying them into seven categories: Angry, Disgust, Fear, Happy, Sad, Surprise, and Neutral.

The project includes data preprocessing, CNN model training, and live emotion prediction. After training, the saved model is loaded to perform real-time detection on faces captured through a camera feed.

1) add fre2013.csv in data/
2) add the haarcascade_frontalface_default.xml
